# Coastal Kings Football Academy

**Live Site:** https://coastal-kings-website.vercel.app/

## 📋 Project Overview
Production website for Coastal Kings Football Academy (Coach Simon). 
Modern registration funnel with cloud backend infrastructure.

## 🚀 Tech Stack
- **Frontend:** Next.js 16 (App Router), React 19, TypeScript, Tailwind CSS 4
- **Animations:** Framer Motion
- **Backend (In Progress):** AWS Lambda, API Gateway, DynamoDB
- **Infrastructure:** Terraform (IaC)
- **Deployment:** Vercel (Frontend), AWS (Backend)
- **Testing:** Jest, React Testing Library
